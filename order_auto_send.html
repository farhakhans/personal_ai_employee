Order System - Auto Send to Android

Setup Instructions (Zaroori Hai!)
Android par messages automatically bhejne ke liye:
  1. Step 1: Apne Android phone mein WhatsApp install karein
  2. Step 2: Pushbullet app install karein (Play Store se)
  3. Step 3: Pushbullet mein sign karein (Google account se)
  4. Step 4: PC par bhi Pushbullet Chrome extension install karein
  5. Step 5: Neeche diya gaya form fill karein aur order create karein
  6. Step 6: Order details Android par notification ke through aa jayengi!
Alternative: Agar Pushbullet nahi use karna, toh "Send via WhatsApp" button use karein. WhatsApp Web khulega, aap manually send kar sakte hain.

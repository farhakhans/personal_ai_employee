## 🚀 Getting Started - Step by Step:

### Step 1: Install WhatsApp Business (2 minutes)
```
1. Open Play Store
2. Search: "WhatsApp Business"
3. Install
4. Open and verify your number
```

### Step 2: Setup Quick Replies (3 minutes)
```
1. Open WhatsApp Business
2. ⋮ (3 dots) → Business tools
3. Quick replies → + Add
4. Create these:

Shortcut: /payment
Message: 🎉 Payment Received!
Thank you for your payment of Rs. {amount}.
Your account has been credited. 🙏

Shortcut: /order
Message: ✅ Order Confirmed!
Order ID: {order_id}
Total: Rs. {total}
Delivery: 3-5 business days 📦

Shortcut: /reminder
Message: 🔔 Payment Reminder
Amount Due: Rs. {amount}
Due Date: {due_date}
Please process payment. Thank you! 💰
```

### Step 3: Enable Away Message (2 minutes)
```
1. Business tools → Away message
2. Turn ON: "Send away message"
3. Message: "Thanks for contacting us! We'll reply within 1 hour."
4. Schedule: Always send (or set business hours)
```

### Step 4: Setup Labels (2 minutes)
```
1. Business tools → Labels
2. Create:
   - 🟢 New Order
   - 🔵 Payment Pending
   - 🟡 Payment Received
   - 🟣 Completed
3. Tag conversations by swiping left
```

### ✅ Done! Your Android WhatsApp is automated!

```
Customer: "I sent payment Rs. 2500"
↓
You: Type "/payment"
↓
Auto-sends: 🎉 Payment Received! Thank you...

```

### Sending Payment Request:
```
1. Open WhatsApp chat
2. Type: "/payment"
3. Fill amount: 2500
4. Send!
```

### Order Confirmation:
```
1. Open customer chat
2. Type: "/order"
3. Fill: Order ID, Amount
4. Send!
```

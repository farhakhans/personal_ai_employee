Quick Setup Guide

LinkedIn Setup
  1. Go to LinkedIn Developer Portal
  2. Create a new app
  3. Get your OAuth 2.0 access token
  4. Get your Person URN from API response
  5. Token expires in 60 days
Facebook & Instagram Setup
  1. Go to Graph API Explorer
  2. Select your app and get Page Access Token
  3. Copy your Page ID from About section
  4. For Instagram, get Business Account ID
  5. Token expires in 60 days
Tip: Save this page after configuration. Your credentials are stored securely in the .env file.

# 📱 WhatsApp Watcher - CLI Commands

## Run WhatsApp Watcher Individually

### **Method 1: Batch File (Easiest)**
```bash
START_WHATSAPP.bat
```

### **Method 2: Python Script**
```bash
python run_whatsapp_watcher.py
```

### **Method 3: Direct Python Command**
```bash
python AI_Employee_System\Watchers\start_whatsapp_watcher.py
```

---

## What It Does:

- ✅ Monitors WhatsApp messages
- ✅ Captures incoming chats
- ✅ Auto-replies (if configured)
- ✅ Saves messages to vault

---

## Verify Running:

Check console output for:
```
✅ WhatsApp Watcher started
🟢 Status: Running
```

---

## Stop WhatsApp Watcher:

**Method 1:** Close console window

**Method 2:** Press `Ctrl+C` in console

---

## Quick Start:

```bash
START_WHATSAPP.bat
```
